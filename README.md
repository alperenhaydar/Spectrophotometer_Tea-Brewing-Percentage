# Spectrophotometer_Tea-Brewing-Percentage
This project designs a portable spectrophotometer system that enables the analysis of the tea brewing process using optical methods. The system measures the light intensity passing through the sample using a 525 nm wavelength LED light source and an opposing photodetector.

The amount of light absorbed by the sample is converted into digital data via an ADC through a microcontroller to determine the tea's density and brewing level.
